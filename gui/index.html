UART

Połączenie PC Agent - STM32

Agent utrzymuje port COM, sygnał życia i parser ramek TEL/RSP/EVT.

Port szeregowy COM?

Ustawiany przez MIKROTOM_SERIAL_PORT albo argument run_agent.bat COMx.

Prędkość transmisji 115200

Domyślnie 115200, zgodnie ze skryptem Agenta.
